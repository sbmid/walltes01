Anonymous Chat API

The backend foundation for anonymous chatting. Explore and test endpoints live!

POST /api/chat/match

Find a partner or enter the waiting queue. Requires user_id.

Code Example
fetch('/api/chat/match', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ user_id: 'user_123' })
})

POST /api/chat/poll

Poll for room status and new messages. Requires room_id, user_id.

Code Example
fetch('/api/chat/poll', {
  method: 'POST',
  body: JSON.stringify({ room_id: 1, user_id: 'user_123' })
})
