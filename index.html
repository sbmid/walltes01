AnonymousChat

API Reference

Integrate secure, serverless anonymous chatting into your applications with our robust REST API.

POST /api/chat/match

Initiates a match. If an empty room exists, the user is paired. Otherwise, a new waiting room is created.

Parameters
user_idstring, required
Unique identifier for the client making the request.
cURL Example
curl -X POST https://apianon.vercel.app/api/chat/match \ -H "Content-Type: application/json" \ -d '{ "user_id": "user_123" }'

                    POST
                    /api/chat/poll
                

                
Polls the server to check the room's status and retrieve any new messages.

                
                

                    
Parameters

                    
                        

                            
                                room_idstring, required
                                
The UUID of the active chat room.

                            		
                        

                        

                            
                                user_idstring, required
                                
Unique identifier for the client.

                            		
                        

                    
                

            

            
            

                
cURL Example

                
curl -X POST https://apianon.vercel.app/api/chat/poll \
  -H "Content-Type: application/json" \

                    POST
                    /api/chat/send
                

                
Dispatches a text message payload to the specified active room.

                
                

                    
Parameters

                    
                        

                            
                                room_idstring, required
                            
                        

                        

                            
                                sender_idstring, required
                            
                        

                        

                            
                                messagestring, required
                                
The text content of the message.

                            		
                        

                    
                

            

            
            

                
cURL Example

                
curl -X POST https://apianon.vercel.app/api/chat/send \
  -H "Content-Type: application/json" \
  -d '{ "room_id": "uuid", "sender_id": "user_123", "message": "Hi!" }'
