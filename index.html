Anonymous Chat API

A lightweight, scalable anonymous chat backend.

🇬🇧 English Documentation

Welcome to the Anonymous Chat API. All endpoints accept POST requests with JSON payload.

Endpoints

POST /api/chat/match
Find a partner or enter the waiting queue. Requires user_id.

POST /api/chat/poll
Check for new messages and room status. Requires room_id, user_id, and optional last_message_id.

POST /api/chat/send
Send a message to the active room. Requires room_id, sender_id, and message.

POST /api/chat/leave
Leave and close the current chat room. Requires room_id and user_id.

نقاط النهاية (Endpoints)

POST /api/chat/match

التحقق من الرسائل الجديدة وحالة الغرفة. يتطلب room_id و user_id، و last_message_id (اختياري).

POST /api/chat/leave
